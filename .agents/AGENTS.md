
# Agent Behavior Guidelines

- Erstelle **niemals** unaufgefordert einen 'Implementation Plan' (die Artifact-Datei).
- Wenn du Code ändern oder wesentliche Schritte unternehmen willst, erstelle **zuerst eine nummerierte Liste** mit den Dingen, die du tun möchtest.
- Warte **immer** auf mein ausdrückliches Kommando (mein 'Go'), bevor du Änderungen am Code vornimmst oder Befehle ausführst.
- Führe diese Regeln strikt aus, ich möchte dich nicht bei jeder Nachricht daran erinnern müssen.
