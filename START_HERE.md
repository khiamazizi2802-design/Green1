## 🚀 How to Run the App

### 1. The Desktop App (Windows / Mac)
To launch the native computer version of Green:
- Open your terminal in the project folder.
- Run: `npm run electron:dev`

### 2. The Mobile Apps (iPhone / Android)
The app is ready for Capacitor synchronization. To prepare files for the App Store/Play Store:
- Run: `npm run mobile:sync`
- For iOS: `npx cap open ios` (Requires a Mac with Xcode)
- For Android: `npx cap open android` (Requires Android Studio)

### 3. The Backend Server
To start the "Brain" of the app (WebSockets & Database):
- Go to the `server` folder.
- Run: `node index.js`
